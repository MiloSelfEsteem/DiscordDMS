Dead mans switch code for python discord bots (DiscordDMS).

Super simple code to implement into any discord bot for a "dead mans switch".

It was either this or "self botting" but I do not want it to break discord ToS.

Checks every hour as to see if a specific user (ideally the server owner) has sent a message, if silence after some time, this code will make the bot send a message into a specific channel (ideally announcements).

To use:

Make sure the bot can run 24/7, either on your own systems or externally, you should know this by now.

Paste in everything in the config section, alter to your liking.

Paste in the rest of the code.

It will check every hour if you have sent a message.
